    33    /*
    34     *  ======== Platform.xdc ========
    35     *  This module implements xdc.IPlatform and defines configuration
    36     *  parameters that correspond to this platform's Cpu's, Board's, etc.
    37     *
    38     *  The configuration parameters are initialized in this package's
    39     *  configuration script (package.cfg) and "bound" to the TCOM object
    40     *  model.  Once they are part of the model, these parameters are
    41     *  queried by a program's configuration script.
    42     *
    43     *  This particular platform has a single Cpu, and therefore, only
    44     *  declares a single CPU configuration object.  Multi-CPU platforms
    45     *  would declare multiple Cpu configuration parameters (one per
    46     *  platform CPU).
    47     *
    48     */
